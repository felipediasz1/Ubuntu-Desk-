1. Instalação Silenciosa (Windows)

Execute o instalador sem janelas. O servidor é configurado automaticamente no cliente durante a instalação.

Comando completo com servidor pré-configurado:
ubuntu-desk-setup.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART /SERVER={{ server_ip or 'IP_DO_SERVIDOR' }} /KEY={{ pub_key or 'CHAVE_PUBLICA' }}
Mínimo (sem configuração de servidor):
ubuntu-desk-setup.exe /VERYSILENT /SUPPRESSMSGBOXES /NORESTART
2. Script PowerShell — Deploy em Massa

Use build\deploy_windows.ps1 para automatizar o deploy em múltiplas máquinas. Suporte a caminho UNC (\\servidor\share).

Deploy local silencioso:
.\build\deploy_windows.ps1 -ServerIP {{ server_ip or 'IP_DO_SERVIDOR' }} -PubKey "{{ pub_key or 'CHAVE_PUBLICA' }}" -Silent
Deploy via UNC com startup automático:
\\servidor\share\deploy_windows.ps1 -InstallerPath \\servidor\share\ubuntu-desk-setup.exe -ServerIP {{ server_ip or 'IP_DO_SERVIDOR' }} -PubKey "{{ pub_key or 'CHAVE_PUBLICA' }}" -Silent -Startup
Apenas atualizar servidor em máquina já instalada:
.\build\deploy_windows.ps1 -ServerIP {{ server_ip or 'IP_DO_SERVIDOR' }} -PubKey "{{ pub_key or 'CHAVE_PUBLICA' }}" -ConfigOnly
3. Group Policy Object (GPO)

Configure um GPO de Inicialização de Computador para instalar automaticamente em todas as máquinas do domínio.

  1. Copie installer\Output\ubuntu-desk-setup.exe e build\deploy_windows.ps1 para um compartilhamento de rede (ex: \\ad\SYSVOL\scripts\ubuntudesk\)
  2. Abra Gerenciamento de Política de Grupo → crie/edite um GPO para a OU desejada
  3. Vá em: Configuração do Computador → Configurações do Windows → Scripts → Inicialização
  4. Clique em Adicionar e configure:
Programa:
powershell.exe
Parâmetros:
-ExecutionPolicy Bypass -File "\\servidor\scripts\ubuntudesk\deploy_windows.ps1" -ServerIP {{ server_ip or 'IP_DO_SERVIDOR' }} -PubKey "{{ pub_key or 'CHAVE_PUBLICA' }}" -Silent -Startup
4. Executável com servidor no nome (sem instalador)

Renomear o executável com o padrão ubuntu-desk-host=IP,key=CHAVE.exe configura automaticamente o servidor ao ser executado pela primeira vez. Útil para distribuição direta sem instalador.

Nome do arquivo:
ubuntu-desk-host={{ server_ip or 'IP_DO_SERVIDOR' }},key={{ pub_key or 'CHAVE_PUBLICA' }}.exe
